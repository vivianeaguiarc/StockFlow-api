# StockFlow API

## Visão Geral

Sistema SaaS de Gestão de Estoque Multiempresa.

Objetivo:
Desenvolver uma API REST profissional para portfólio demonstrando:

- Node.js
- TypeScript
- PostgreSQL
- Prisma
- JWT
- Docker
- Testes
- CI/CD
- Clean Architecture
- Segurança
